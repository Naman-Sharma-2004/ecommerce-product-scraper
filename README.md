# 🛒 E-commerce Product Scraper

A Python-based web scraper that extracts product data from an e-commerce website using BeautifulSoup and Requests.

## 🚀 Features

* Scrapes multiple pages (pagination support)
* Extracts:

  * Product Name
  * Price (converted to numeric)
  * Rating
  * Availability
  * Product Link
* Data cleaning and preprocessing
* Exports data to:

  * CSV
  * JSON
  * Excel

## 🛠️ Tech Stack

* Python
* Requests
* BeautifulSoup
* Pandas

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python scraper.py

## 📊 Output

Clean dataset ready for analysis and machine learning.

## 💡 Highlights

* Handles pagination efficiently
* Converts raw scraped data into a structured format
* Implements error handling and data validation

## 📌 Note

Built using a controlled scraping environment and designed to be adaptable to real-world websites.
